## Задание 2
Программа работает только с текущим пользователем, считывая пользовательские переменные среды. Комментарий дан ниже.

Так как в задании было четко написано "чтение и изменение некоего набора переменных среды" и далее по тексту было сказано "значения должны быть доступны другим приложениям", то для данного проекта подходит переменная среды Windows.
Одно из её ограничений, что в сумме длина всех значений не может быть больше 32к символов. Но для тестового задания этого было бы достаточно.
Как вариант, использовать для переменной среды некий файл, который уже не имеет никаких ограниений по длине.

Комментарии сохраняются в ветку реестра. По умолчанию "HKCU/Software/EnvComments", но можно изменить на другой путь, указав его в appsettings.json в переменной RegPath объекта EnvVarSettings. В этом же объекте можно указать переменные перечислив их в Variables.

Можно было бы считывать системные переменные среды, требуя админ права для запуска приложения и поменяв в сервисе EnvVarService EnvironmentVariableTarget.User на EnvironmentVariableTarget.Machine. Аналогично и с веткой реестра - можно было бы заменить на LocalMachine.
